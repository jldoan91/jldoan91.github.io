Introduction

Python is an interpreted language for general-purpose programming. It first released in 1991 and emphasizes code readability. Pythono and CPython are managed by Python Software Foundation, which is a non-profit organization.

Python was created by Guido van Rossum in the late 1980s. It was first implemented in December of 1989 as a successor to the ABC language.

Here's a little sample of what Python code looks like.

          return abs( sum(i * j for i, j in zip(x, y[1:])) + x[-1] * y[0]

Python is dynamically typed so you don't have to define the data type as it will guess based on the value the variable contains.

For e.g. 

          
            myvar = "Hello Python"
In this code myvar is a string.

If you were to type this code later 

          
            myvar = 1
myvar would now be an int rather than string

Getting Started

You can check if you have Python installed on windows by opening the command line (cmd.exe) and entering the following 

          
            C:\Users\Your Name>python --version
If you're on a Mac or Linux you can enter the following in terminal/command line 
          
            python --version

Most PCs and MACs come installed with Python. However if you find that you do not have it installed on your computer you can download it here.

In order to run a Python file you'll go into the command line and enter "python filename.py" below is an example. 

          
            C:\Users\Your Name>python helloworld.py

You can enter the Python command line by entering "python" into the terminal/command line application

Once in to the Python command line you can write any python and it will run similar to the console log in web browsers for JavaScript.
